🎯 DIRETRIZES DE IMPLEMENTAÇÃO

Responsividade por Versão

LUXURY Mobile:

  • • Mantém minimalismo com gestos touch
  • • Imagens em tela cheia com swipe
  • • Navegação oculta em menu discreto
  • • Tipografia reduzida proporcionalmente

MODERNA Mobile:

  • • Interface mobile-first nativa
  • • Bottom navigation com ícones
  • • Cards em carrossel horizontal
  • • Filtros em modal fullscreen

CLÁSSICA Mobile:

  • • Adaptação cuidadosa mantendo todas informações
  • • Menu hamburguer tradicional
  • • Tabelas convertidas em cards empilhados
  • • Formulários em página dedicada

Acessibilidade

Todas as versões devem incluir:

  • • Contraste WCAG AAA entre texto e fundo
  • • Alt text descritivo em todas imagens
  • • Navegação completa por teclado
  • • Aria labels em elementos interativos
  • • Opção de alto contraste (especialmente importante na versão Clássica)

Performance

LUXURY:

  • • Lazy loading prioritário para imagens grandes
  • • Preload de fontes decorativas
  • • Animações com CSS transforms apenas

MODERNA:

  • • Progressive Web App capabilities
  • • Service worker para offline browsing
  • • Imagens em formato WebP com fallback

CLÁSSICA:

  • • HTML semântico simples
  • • Mínimo de JavaScript
  • • Compatibilidade com IE11+
